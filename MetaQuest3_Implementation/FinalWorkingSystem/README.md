# Final Interactive AR Chess Board Game Working System - Ane O'Byrne & Diya Mathew
**Final working interactive chess game** featuring **grabbable, movable chess pieces** in a fully immersive **AR environment**. Users can interact with both white and black pieces, move them freely, and experience a set of in-game features, all within the augmented reality board.

---

## Features

- **Augmented Reality Chess Board**  
  Spawn a full 3D chessboard in your AR environment.

- **Interactable Chess Pieces**  
  Move both white and black pieces with touch/gesture-based interactions.

- **Immersive Interfaces**  
  - Option to expand to adding different board games as future implementation. 
  - View instructions on how to play  
  - Access in-game buttons for additional actions:
    - Review rules
    - Swap players
    - Rotate board
    - Reset board
    - Exit to main menu

- **Smooth Gameplay Experience**  
  Intuitive controls and interactive buttons for a complete AR chess experience.
